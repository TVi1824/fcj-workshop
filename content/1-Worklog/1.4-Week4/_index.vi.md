### Kết quả đạt được tuần 4:

* Cấu hình thành công các kịch bản tự động hóa và giám sát hệ thống với Lambda, Cloudwatch và Grafana.

* Nắm vững các công cụ quản trị hệ thống tập trung như Tagging, Resource Groups và AWS Systems Manager.

* Thiết lập thành công quản lý định danh tập trung (Amazon SSO) và phân quyền chặt chẽ với IAM Permission Boundary.

* Triển khai hệ thống bảo vệ ứng dụng, API với AWS WAF và mã hóa dữ liệu với AWS KMS, tuân thủ đánh giá từ Security Hub.
